SEARCH & DESTROY COMPANION — OPERATOR ATLAS 2.0

WHAT THIS BUILD IS
------------------
A map-first, mobile Search & Destroy companion designed for a squad leader to open during map voting, select the round situation, and brief five operators before the round begins.

VERIFIED MAP SCOPE
------------------
Official 2026 competitive Search & Destroy pool:
- Tunisia
- Firing Range
- Coastal
- Slums
- Meltdown

Additional requested Ranked reference:
- Raid

Activision's public Season 6 announcement confirms the new Ranked season but does not publish a complete mode-by-mode Ranked map list. This build therefore labels only the five official competitive S&D maps as verified and does not disguise guesses as an official Ranked rotation.

CORE FEATURES
-------------
- Visual map selector as the opening screen
- Six locally packaged tactical maps that work offline
- Overview, Attack A, Attack B, Defense, Retake, and Sightline views
- Toggleable callouts, bomb sites, spawns, routes, and sightlines
- Tap-to-open position intel: what it is, what it watches, and how to clear it
- Team-leader call and five operator assignments per situation
- Expand, zoom, and pan map controls
- Map-specific squad notes stored on the device
- Existing 30-operation Field Manual
- Installable PWA and offline cache

UPDATE THE EXISTING GITHUB PAGES APP
------------------------------------
1. Extract this ZIP.
2. Open the existing search-and-destroy-companion GitHub repository.
3. Choose Add file > Upload files.
4. Upload everything inside the extracted folder, including the maps folder.
5. Let GitHub replace matching files.
6. Commit to main with: Replace map module with Operator Atlas 2.0
7. Allow GitHub Pages a few minutes to rebuild.
8. If the installed phone app still shows the old version, fully close it and reopen it. A hard refresh or clearing the old site cache may be needed once because the previous service worker was aggressive.

IMPORTANT
---------
Upload the extracted files, not the ZIP itself.
